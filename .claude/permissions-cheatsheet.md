# Дозволені команди — довідка

## Читання файлів і директорій
| Команда | Що робить |
|---------|-----------|
| `ls` | Список файлів у папці |
| `cat` | Вивести вміст файлу |
| `head` | Перші N рядків файлу |
| `tail` | Останні N рядків файлу (або стежити за логами: `tail -f`) |
| `find` | Шукати файли по імені/типу/даті |
| `grep` | Шукати текст у файлах |
| `rg` | Швидкий grep (ripgrep) |
| `wc` | Порахувати рядки/слова/символи у файлі |
| `file` | Визначити тип файлу (текст, бінарний, зображення) |
| `stat` | Детальна інфо про файл (розмір, дата, права) |
| `du` | Розмір папки/файлу на диску |
| `df` | Вільне місце на дисках |
| `pwd` | Показати поточну директорію |
| `which` | Де знаходиться програма (`which python3` → `/usr/bin/python3`) |
| `where` | Як which, але показує всі варіанти |
| `whoami` | Поточний користувач |
| `env` | Всі змінні середовища |
| `printenv` | Вивести конкретну змінну середовища |
| `tree` | Дерево папок і файлів |

## Процеси і порти
| Команда | Що робить |
|---------|-----------|
| `lsof` | Хто використовує файл/порт (`lsof -i:8080`) |
| `kill` | Завершити процес по PID |
| `pkill` | Завершити процес по імені |
| `xargs` | Передати вивід однієї команди як аргумент іншій |
| `fuser` | Хто використовує файл/порт (альтернатива lsof) |
| `ps` | Список запущених процесів |
| `top` / `htop` | Моніторинг процесів в реальному часі |
| `netstat` / `ss` | Мережеві з'єднання і відкриті порти |

## Хуки (автоматичні дії по подіях)
| Подія | Що запускається | Коли спрацьовує |

| `PostToolUse` → `Bash` | `open-pdf-after-typst.sh` | Після `typst compile` — автоматично відкриває згенеровані PDF у Preview |
| `Notification` | `restart.sh` (AIchallangewithlove) | Коли Claude завершив довгу задачу — перезапускає Ktor сервер + Desktop додаток для тестування кожаним мішком |
